# Skillbag-ai GitHub Profile Repository

This directory is a scaffold for the Skillbag-ai organization profile
repository.

## GitHub Repository Name

Create a public repository in the `Skillbag-ai` organization named:

```text
.github
```

GitHub renders `profile/README.md` from that repository on the organization
profile page.

The final repository URL should be:

```text
https://github.com/Skillbag-ai/.github
```

## Files

- [profile/README.md](./profile/README.md): public organization profile

## Suggested Setup

1. Create the `Skillbag-ai/.github` repository on GitHub.
2. Copy this directory's contents into that repository.
3. Commit and push to the default branch.
4. Visit `https://github.com/Skillbag-ai` to verify the profile renders.

Keep the organization profile high-level. Detailed normative rules should stay
in the core `skillbag` repository.
